  ## Description
  The back end handler for an e-commerce website. Organizes and tracks products using categories and tags. Includes cost and stock.

  ## Installation Instructions
  Fork from github, run NPM install, initialize the database via MySQL shell, set up your .env file, run npm start.

  ## Usage Instructions
  Query using Insomnia or a similar request generator.
  
  Find an example of some tests being run [here](https://drive.google.com/file/d/1JBIFtBa9zRJJKZU_6DzEvHKluGJkZvY_/view)
